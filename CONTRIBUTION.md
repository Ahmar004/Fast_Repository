# Contributing to FAST Repository

Thank you for helping the community!!!

## How to Add Resources

1. Fork this repository
2. Go to `/data`
3. Open the relevant course JSON file
4. Add your resource in the correct section:
   - Campus-wise
   - YouTube
   - General
5. Commit changes
6. Create a Pull Request

## Rules
- Only FAST-related academic resources
- No private or paid content
- Links must be accessible
- Be honest and accurate

All submissions are reviewed before merging.
